Events

Event Description
init For when the form is initialized
json-form.update Used to respond to changes in the form.
json-form.update.data Used to respond to changes in the data.
json-form.update.target Used to respond to changes in the target value.

Filters

Method Description
updateTarget( value: string ):string For when you need to change the value that is placed on the target.
updateData( args: { path , value , type }): any For when you want to manipulate data values.
