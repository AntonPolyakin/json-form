Method Description
update() : void Regenerates the form using the current data.
destroy() : void It can be used to destroy the form (All fields will be removed).
data() : any Returns the current data.
raw() : any Returns the orginal data.
reset( model:string = "" ) : any ,
resetAll() : any 		It is used to restore the data to the initial state.
addEventListener( type: string, listener: EventListenerOrEventListenerObject, options?: boolean | AddEventListenerOptions ) : void Attaches an event handler to the form.
model() : string Returns the model.
hasModel() : boolean Checks if the model is set.
